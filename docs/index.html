Quickstart

快速开始

3 步完成所有配置，然后一条命令开始批量上传。

1

克隆仓库 + 首次配置

自动检测 Unity 安装路径，下载 vrc-get 包管理器。

git clone https://github.com/dwgx/VRC-Auto-Uploader.git
cd VRC-Auto-Uploader
python main.py setup
2

首次登录 VRChat（仅需一次）

运行任意一次上传命令后，Unity 会弹出 VRChat SDK 登录界面。手动登录一次，之后 session 会持久化，无需再次输入账密。

python main.py upload --package "D:\Model\Azuki\Azuki.unitypackage"
3

批量上传整个目录

扫描整个模型目录，自动解压、依次上传所有模型，完全无须监督。

python main.py batch --dir "D:\Model" -y
FAQ

常见问题

遇到问题？先在这里查查。

🔐 为什么必须手动登录一次？

VRChat SDK 不支持在代码中直接传递账号密码。首次登录后，SDK 会将 session token 持久化到本地，后续所有上传均自动使用已保存的 token，无需再次登录。

🖥️ 为什么上传时 Unity GUI 会弹出来？

VRChat SDK 的 VRCSdkControlPanel.TryGetBuilder() 依赖 Unity Editor GUI 初始化。在 -batchmode -nographics 下 Builder 实例不会注册，上传必然失败。本工具使用正确的 GUI 模式启动 Unity，但全程自动化无需人工操作。

🖱️ 上传过程中出现弹窗需要点击怎么办？

工具内置 PopupSuppressor 脚本，会自动识别并关闭 VRChat SDK 的版权声明弹窗和更新提示弹窗，无需手动点击。

📦 支持哪些压缩格式？

支持 .zip（使用 Windows 内置 tar 或 PowerShell）、.rar 和 .7z（需要系统安装 7-Zip）。直接放置的 .unitypackage 文件无需解压，直接使用。

🎨 模型显示粉红色材质怎么办？

工具会自动安装 lilToon（日本模型最常用的 Shader）。如果你的模型使用 Poiyomi 等其他 Shader，可能仍会有粉红材质，但 VRChat 官方不允许在无 Transparency 情况下使用 Poiyomi，请使用 lilToon 版本的模型。

⚠️ 上传权限要求是什么？

VRChat 账号需要达到 New User（蓝名） 及以上信任等级才能上传 Avatar。如果你是新账号，需要先在游戏里多玩一段时间。
